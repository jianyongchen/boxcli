# Cisco-like CLI in Rust

A production-ready, modular Cisco IOS-style CLI written in Rust.

## ✨ Features

- Multi-level configuration modes
- Context-aware Tab completion
- `help` and `?` (Cisco-style)
- Command aliases (`conf t`)
- Configuration tree with `show running-config`
- Structured logging (`tracing`)
- Unit tests & CI

## ▶️ Run

```bash
cargo run
